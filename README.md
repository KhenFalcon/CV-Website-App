## Running Locally:
- clone this repository
- open repository in IDE (vscode) or powershell terminal
- (in powershell terminal) enter 'npm install'
- enter 'npm run dev'
- output should contain a local hyperlink (ex: "http://localhost:5173")
- ctrl + left-click on the link or copy-paste it into your browser
